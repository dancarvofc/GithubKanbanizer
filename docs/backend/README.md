# Documentação do backend

Esta pasta reúne documentação didática e acessível do backend do projeto.

- **Objetivo:** apresentar estrutura, arquitetura, fluxo de dados e diagramas para desenvolvedores.
- **Onde:** arquivos nesta pasta e subpastas `mermaid`.

Arquivos principais:

- `structure.md` — visão da estrutura de pastas e responsabilidades.
- `arquitetura.md` — visão arquitetural e responsabilidades dos módulos.
- `sequencia-dados.md` — fluxo de dados (sequência de chamadas) com exemplos.
- `mermaid/class-diagram.mmd` — diagrama de classes (Mermaid).
- `mermaid/sequence-kanban.mmd` — diagrama de sequência (Mermaid).
